Дэлгүүр дээр сар бүрийн үнийн өөрчлөлт дандаа гараар хийгддэг тул тул тухайн барааны борлуулалт, эрэлт хэр байгаа, мөн худалдан авч байгаа үнэ ашиг зэргийг тооцоод боломжит үнэ санал болгодог агент хөгжүүлэх.

Өгөгдсөн өгөгдлийн төрөл: int.csv

1) Борлуулалтын тоо
2) Эрэлтийн тоо хэмжээ
3) Бараа авах үнэ 
4) Ашиг

Гүйцэтгэх ажлуудын дараалал:

- Бизнесийн дүрмүүдийг гаргах
- Яагаад энэхүү үнийн саналыг гаргаж байгаагаа тайлбарлах
- UI дизайны гаргах
- CSV өгөгдлийн бүтцийг гаргах
- API эсвэл model алийг нь ашиглахаа шийдэх
